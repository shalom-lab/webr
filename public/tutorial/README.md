# R 语言入门教程

这个文件夹包含 R 语言入门教程的示例代码文件。

## 文件说明

- `基本数据类型.R` - R 语言的基本数据类型介绍
- `函数.R` - R 函数的定义和使用
- `数据框.R` - 数据框（Data Frame）的操作

## 使用方法

1. 在 Files 面板中，打开 `/home/web_user/tutorial` 文件夹
2. 双击任意 `.R` 文件即可在编辑器中打开
3. 在终端中运行代码或使用编辑器运行代码

## 添加更多教程文件

如果你想添加更多教程文件：

1. 将 `.R` 文件放在项目的 `public/tutorial` 目录下
2. 在 `src/use/webrInit.js` 的 `tutorialFiles` 数组中添加文件名
3. 重新构建项目，文件会自动加载到 WebR 文件系统中
